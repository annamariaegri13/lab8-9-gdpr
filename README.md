# lab8-9-gdpr

# Cookie-uri – Laborator GDPR

## Descriere proiect
Acest proiect are ca scop implementarea și testarea mecanismului de cookie-uri în browser folosind JavaScript (`document.cookie`), conform exercițiilor din MDN Web Docs.

Proiectul evidențiază modul în care cookie-urile pot stoca și gestiona informații despre utilizator și impactul acestora asupra vieții private.

## Funcționalități implementate
- crearea de cookie-uri în browser
- citirea tuturor cookie-urilor active
- extragerea unui cookie specific
- ștergerea cookie-urilor
- simularea unei acțiuni executate o singură dată
- testarea comportamentului cookie-urilor în browser

## Legătura cu protecția datelor
Cookie-urile pot conține identificatori care pot fi considerați date personale conform GDPR. Utilizarea acestora trebuie să respecte principiile de transparență și consimțământ explicit al utilizatorului.

## Testare
Funcționalitatea a fost testată în browser folosind DevTools (Application → Cookies), verificând comportamentul de creare, modificare și ștergere a cookie-urilor.
